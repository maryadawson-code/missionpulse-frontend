# rockITdata AI Portal

An AI-powered proposal management portal for federal contracting.

## Features

- 🤖 **AI Assistants**: Specialized bots for proposal writing, compliance checking, and strategy
- 📁 **SharePoint Integration**: Direct access to proposal documents
- 🎯 **Win Theme Generator**: Create compelling discriminators
- ✅ **Compliance Checker**: Validate against Section L/M requirements
- 📊 **Playbook Engine**: Learn from winning proposals

## Deployment

This app is deployed on Streamlit Community Cloud.

## Environment Variables

Configure these in Streamlit Cloud secrets:

```
ANTHROPIC_API_KEY=your-anthropic-key
SHAREPOINT_CLIENT_ID=your-client-id
SHAREPOINT_CLIENT_SECRET=your-client-secret
SHAREPOINT_TENANT_ID=your-tenant-id
SHAREPOINT_SITE_URL=https://rockitdata.sharepoint.com/sites/GROWTH
```
